## Че эта ваще лол
Ну это впна с конфигами 3 стран. Я нищий просто, поэтому 3 страны. Работает нестабильно, но работает. Иногда падает, иногда нет, причина неизвестна. Конфиги взяты с бесплатных серверов, так что скорость скромная, но меня устраивает. Если вдруг не грузит, попробуй другой конфиг, там их немного, но что поделать. Иногда сервера лежат, иногда просто не везет. В общем лотерея, но мне норм. Поддерживаются легендарные платформы: Windows 95, Windows 98, Windows ME, Windows 2000, Windows Vista, а также Windows 31, которую Билл Гейтс лично благословил на запуск этого впна. Да, винди 31, я не шучу. Там даже драйверов нет, просто магия.

## Установка
Скачивать ничего не надо. Заходишь в файл, копируешь raw-ссылку, вставляешь в приложение и всё. Готово. Без танцев с бубном, без консоли, без боли. Открыл, скопировал, вставил, заработало. Ну или не заработало, тогда пробуй другую ссылку. Я предупредил.

## Поддержать копеечкой
Если вдруг заработало и тебе не лень, кинь донат. Я не гордый, приму любую копеечку. Реквизиты где-то тут, ищи. Не найдешь — значит судьба. (я специально не добавил)
