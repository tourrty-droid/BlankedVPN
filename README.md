## Установка сервера

```bash
git clone https://github.com/tourrty-droid/BlankedVPN.git
cd BlankedVPN
docker compose up -d
```

Подключение

1. Скачай конфиг из панели управления
2. Импортируй в клиент (WireGuard, NekoBox, Streisand)
3. Подключись
